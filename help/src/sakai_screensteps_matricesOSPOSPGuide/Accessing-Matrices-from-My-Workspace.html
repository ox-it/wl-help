Accessing Matrices from My Dashboard

When the Matrices tool is added to My Dashboard, it displays a list of all of the matrices to which you have access. Using this tool, you can open any matrix in the list without visiting the site in which the matrix is located. To turn on the Matrices tool in My Dashboard, follow these steps:

  1. Click on the Site Info tool.
  2. Click on the Page Order link located on the Site Info menu bar.
  3. Click on the Add page(s) to your site link located on the Site Info menu bar.
  4. Check the box next to the Matrices tool.
  5. Verify that the Matrices tool has been added and is in your preferred page order.
  6. Click on the Save button.
  7. Once the Matrices tool has been added to your My Dashboard, click Matrices to see all of the matrices to which you have access.
  8. To work with a matrix in the list, click its title.

(Note: Different institutions may or may not enable the Matrices tool in My Dashboard. If the Matrices tool has not been enabled in My Dashboard, if Site Info is not a tool in My Dashboard, or if the Matrices tool does not appear under Edit Tools in My Dashboard > Site Info, then you will need to ask your system admin to modify the template for your My Dashboard site. Additionally, the Matrices tool by default only appears in My Dashboard for maintain-level users.)
