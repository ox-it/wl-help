Which browser should I use?

Note: To determine which version of Internet Explorer or Firefox you are using on a Windows computer, in the browser, click Help, and then click About Internet Explorer or About Mozilla Firefox. On a Macintosh, from the Firefox menu, select About Mozilla Firefox.

Use one of the following browsers:

Windows

  • Mozilla Firefox 2.0 and newer
  • Internet Explorer 7.0 and newer

    Note: In Internet Explorer 8, some pages may not display correctly. You may be able to improve the appearance by turning on Compatibility View. Within Internet Explorer, click Tools and then Compatibility View. See this Windows help article for more information.

Macintosh

  • Mozilla Firefox 2.0 and newer

Other platforms

  • Mozilla Firefox 2.0 and newer

Other considerations

Note: You may get unexpected results if you use two browser windows to access the same Sakai 2.6 tool at the same time. For best results, use one browser window to work in Sakai 2.6.
